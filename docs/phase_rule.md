# 阶段表规则

## 变量说明

* 规则里提到的变量都必须可以在设置面板设置，说明里会提供默认值

## 规则

* 5x5的表，每一行代表一个阶段，每个阶段的任务有对应的积分 A[n] (默认为A[5] = {2, 2, 4, 4, 6})
* 每个阶段5个任务，允许双方重复完成任务，但是后完成任务者获取的积分会减少为 B[n] (默认为B[5] = {1, 1, 2, 2, 3})
* 从第一阶段开始，每个阶段每个玩家最多可以完成 C(默认C=3) 个格子
* 玩家完成 D(默认D=2) 个格子后才能去染色下一个阶段的格子
* 玩家可以完成5条竖线以及两个对角线共七种可能性的Bingo，只有第一次完成Bingo者可以获得Bingo积分 E(默认E=3)，全局只能结算这第一次Bingo积分
* 完成第五阶段至少两个格子后允许玩家触发完赛结算，先结算者可以获得积分 F(默认F=3)
  * 玩家可以对自己进行结算操作，裁判也可以进行结算操作，第二个玩家结算后游戏结束
* 游戏结束后积分高的一方获胜，但是如果分数相同，则先结算的玩家获胜
